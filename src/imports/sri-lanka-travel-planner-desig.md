Design a modern responsive web application UI called **“Sri Lanka Travel Planner”**.

Purpose: Help tourists plan trips across Sri Lanka by selecting destinations, generating itineraries, viewing maps, checking weather, and estimating travel budgets.

Design Style:

* Modern travel dashboard
* Tropical color palette (Ocean Blue, Emerald Green, Sand Beige)
* Minimal UI with soft shadows and rounded cards
* Large travel images
* Clean typography (Poppins for headings, Inter for body text)
* Responsive design for desktop and mobile

Create the following pages:

1. **Landing Page**

* Full width hero section with a scenic Sri Lanka image
* Title: “Plan Your Dream Trip Across Sri Lanka”
* Subtitle: Discover beaches, mountains, wildlife, and heritage sites
* Buttons: “Start Planning” and “Explore Destinations”
* Section for Popular Destinations (Sigiriya, Ella, Galle, Kandy, Mirissa)
* Feature cards showing:

  * Smart itinerary generator
  * Live weather updates
  * Budget estimation

2. **Explore Destinations Page**

* Search bar at the top
* Filter chips (Beaches, Mountains, Cultural, Wildlife, Adventure)
* Grid of destination cards
* Each card contains:

  * Destination image
  * Name
  * Location
  * Rating
  * Short description
  * “Add to Trip” button
* Right sidebar showing selected trip destinations

3. **Trip Planner Page**

* 3-column dashboard layout
  Left Panel:
* Form inputs:

  * Number of travel days
  * Travel style (Budget, Standard, Luxury)
  * Number of travelers
  * Preferred categories
    Center Panel:
* Interactive map showing selected destinations
* Travel route between locations
  Right Panel:
* Trip summary card with total days, selected places, and estimated cost

4. **Itinerary Page**

* Timeline layout
* Day-by-day trip plan cards
  Example:
  Day 1 – Colombo to Kandy
  Morning: Arrive Colombo
  Afternoon: Travel to Kandy
  Evening: Temple of the Tooth visit
* Show weather and travel time for each day

5. **Budget Estimator Page**

* Form inputs:

  * Number of travelers
  * Transport type
  * Accommodation type
  * Food budget level
* Display cost breakdown cards:

  * Transport
  * Accommodation
  * Meals
  * Activities
* Large highlighted card showing total estimated travel cost

6. **Trip Summary Dashboard**

* KPI cards showing:

  * Total destinations
  * Total travel days
  * Estimated budget
  * Average weather
* Route map
* Budget pie chart
* Recommended travel tips

Reusable Components:

* Navbar
* Destination Card
* Trip Sidebar
* Map View
* Budget Card
* KPI Stats Card
* Itinerary Timeline Card
* Footer

Include micro-interactions:

* Hover effects on destination cards
* Smooth page transitions
* Animated map markers
* Notification toast when destination is added to trip

Make the design visually rich, travel-themed, and suitable for a modern tourism planning web application.
